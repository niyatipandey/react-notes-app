# 📝 React Notes App

A modern sticky notes app built with React.js and Tailwind CSS.

Users can create, edit, pin, search, and trash notes with local storage support, toast notifications, dark/light mode, and a responsive UI.

---

## ✨ Features

- ➕ Add Notes
- ✏ Edit Notes
- 📌 Pin Important Notes
- 🔍 Search Notes
- 🗑 Move Notes to Trash
- 💾 Local Storage Support
- 🌙 Dark / ☀ Light Mode
- 🔔 Toast Notifications
- 📱 Responsive Design

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Local Storage API
- Lucide React Icons
